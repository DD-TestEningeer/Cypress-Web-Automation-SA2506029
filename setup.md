**Title:** Steps to Install and Run Cypress

**Steps:**

1. Create an empty folder on your PC or laptop.
2. Open the same folder in **Visual Studio Code**.
3. Install Cypress using the terminal with the following command:

   ```
   npm install cypress --save-dev
   ```
4. Open Cypress using:

   ```
   npx cypress open
   ```
5. Click on **End-to-End Testing**.
6. Click on the **Continue** button to set up the configuration files.
7. Choose a browser (e.g., **Chrome**).
8. Click on the **Start End-to-End Testing** button â€” this will open the Chrome browser.
9. Click on **Sample Examples** to view sample tests.
10. Run the sample test file named **todo.cy.js**.
